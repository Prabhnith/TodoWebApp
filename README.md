# TODO WEB APP
A Simple TODO web app made in React with the localstorage and database backup features.

##QuickStart

To install the TODO app:
* 1. git clone https://github.com/Prabhnith/TodoWebApp.git
* 2. cd TodoWebApp
* 3. npm install
* 4. npm start
* 5. check out on localhost:8080
